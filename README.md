# HR Analytics: Employee Attrition Analysis

This project analyzes employee attrition using HR, survey, manager, and attendance data for 4,410 employees. The goal was to identify patterns associated with employee turnover and understand how factors such as job satisfaction, income, promotion timing, and attendance behavior relate to retention risk. :contentReference[oaicite:2]{index=2}

## Project Overview

Employee attrition can affect productivity, hiring costs, and long-term organizational performance. In this project, I built an HR analytics dataset by combining multiple internal data sources and then used exploratory and statistical analysis to study which employee characteristics were most associated with turnover.

The analysis focused on:
- building a combined HR dataset from multiple sources
- engineering attendance and work-hour features from time-log data
- identifying variables associated with employee attrition
- interpreting turnover patterns through statistical analysis :contentReference[oaicite:3]{index=3}
